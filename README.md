# AIBook — AI Language Learning Reader

> **v1.0.0** · Mobile-first PWA · Next.js 16 · Gemini AI

Умное приложение для изучения иностранных языков через чтение книг. Откройте книгу, нажмите на любое слово — и получите мгновенный разбор: перевод, грамматика, контекст, примеры. Сохраняйте слова в карточки и повторяйте.

---

## 📱 Скриншоты

| Главная | Читалка | AI-панель | Карточки |
|--------|---------|-----------|---------|
| Дашборд с прогрессом | Текст книги с подсветкой | Разбор слова/фразы | Список сохранённых слов |

## ✨ Возможности v1.0.0

### 📚 Библиотека книг
- Загрузка файлов форматов **TXT** и **EPUB** с устройства
- Автоматический парсинг и разбивка текста на абзацы
- Определение языка книги по имени файла (`_de`, `-en`, `-fr`, `-es`)
- Хранение всех книг в браузере (localStorage) — без облака, без регистрации
- Удаление книг из библиотеки
- Цветные обложки для визуального различия книг

### 📖 Читалка
- Постабзацный режим чтения — по одному абзацу за раз
- Навигация вперёд/назад с автосохранением прогресса
- Подсветка выбранного слова при нажатии
- Прогресс-бар с процентом прочитанного
- Кнопка возврата в главное меню без потери места

### 🤖 AI-анализ слов (Gemini API)
- **Нажатие на слово** → полный разбор через Google Gemini:
  - Словарная форма (лемма) + часть речи
  - Грамматический род (для немецкого: der/die/das)
  - Перевод слова на родной язык пользователя
  - Контекстное объяснение (1-2 предложения)
- **Фраза**: автовыделение устойчивого словосочетания с типом (идиома / коллокация / составное слово)
- **Предложение**: полный перевод + грамматическая заметка + описание структуры
- **5 примеров использования** слова в живом языке
- Кэширование результатов в localStorage (24 ч) — экономия запросов к API
- Поддержка собственного Gemini API ключа через настройки

### 🗂 Карточки (Flashcards)
- Сохранение слова/фразы/предложения в карточку одним кликом прямо из AI-панели
- Список всех карточек с датой добавления и источником (книга → слово)
- Статус карточки: `new` / `due` / `learning`
- Все карточки хранятся локально в браузере

### ⚙️ Настройки
- Выбор **родного языка** (UI и объяснения)
- Выбор **изучаемого языка** (язык книг для анализа)
- Ввод собственного **Gemini API ключа** (хранится только в localStorage)
- Статистика: минуты чтения, книг начато/завершено, сохранено карточек

### 🏠 Главный дашборд
- Виджет "Продолжить чтение" с последней открытой книгой
- Быстрый переход в библиотеку и в карточки
- Общая статистика профиля

## 🚀 Быстрый старт

### 1. Клонировать репозиторий

```bash
git clone https://github.com/YOUR_USERNAME/aibook.git
cd aibook
```

### 2. Установить зависимости

```bash
npm install
```

### 3. Настроить переменные окружения

```bash
cp .env.example .env.local
```

Отредактируйте `.env.local`:

```env
GEMINI_API_KEY=ваш_ключ_gemini
# Опционально — для будущей облачной синхронизации:
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
```

Получить Gemini API Key: https://aistudio.google.com/apikey

### 4. Запустить в режиме разработки

```bash
npm run dev
```

Откройте http://localhost:3000 в браузере (или на мобильном устройстве в той же WiFi-сети).

## 📖 Как пользоваться

1. **Добавить книгу**: вкладка «Книги» → иконка «+» → выбрать `.txt` или `.epub` файл
2. **Читать**: нажать на книгу → откроется читалка с первым абзацем
3. **Анализировать слово**: нажать на любое слово → снизу появится AI-панель с разбором
4. **Сохранить карточку**: в AI-панели нажать «Сохранить» рядом с нужным элементом (слово / фраза / предложение)
5. **Повторять**: вкладка «Карточки» — все сохранённые слова
6. **Настройки**: вкладка «Настройки» — изменить языки или добавить свой API-ключ

## 🔒 Конфиденциальность

- **Все данные хранятся локально** в вашем браузере (localStorage)
- API-ключ Gemini хранится только в localStorage вашего устройства
- При анализе слова текст текущего предложения и соседних (контекст) отправляется на Gemini API
- Облачная синхронизация (Supabase) — в планах на v2
